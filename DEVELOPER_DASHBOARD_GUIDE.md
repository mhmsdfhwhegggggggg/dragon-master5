# Dragon Telegram Pro - Developer Dashboard Guide

## نظرة عامة

لوحة تحكم المطور هي منصة إدارية شاملة تسمح للمطورين بالتحكم الكامل في نظام إدارة التراخيص والتصاريح.

### رابط الوصول:
```
https://dragon-admin.manus.space
```

### بيانات الدخول:
- البريد الإلكتروني: admin@dragaontelegram.pro
- كلمة المرور: [يتم إرسالها عبر البريد الإلكتروني]

## المميزات الرئيسية

### 1. لوحة التحكم الرئيسية (Dashboard)

#### الإحصائيات:
- **إجمالي المستخدمين**: عدد المستخدمين المسجلين
- **التراخيص النشطة**: عدد التراخيص الفعالة حالياً
- **حسابات Telegram**: عدد حسابات Telegram المتصلة
- **العمليات اليومية**: عدد العمليات المنفذة اليوم

#### الرسوم البيانية:
- **نظرة عامة على العمليات**: آخر 7 أيام
- **نشاط المستخدمين**: المستخدمون النشطون يومياً

### 2. إدارة التراخيص (Licenses)

#### إنشاء ترخيص جديد:
```
1. انقر على "Create New License"
2. اختر نوع الترخيص:
   - Basic: 5 حسابات، 100 عملية/يوم
   - Premium: 10 حسابات، 500 عملية/يوم
   - Enterprise: 50 حساب، 2000 عملية/يوم
3. اختر المستخدم
4. انقر على "Create"
```

#### تفعيل/تعطيل ترخيص:
```
1. اختر الترخيص من القائمة
2. انقر على "Activate" أو "Deactivate"
3. أكد الإجراء
```

#### حذف ترخيص:
```
1. اختر الترخيص
2. انقر على "Delete"
3. أكد الحذف
```

### 3. إدارة المستخدمين (Users)

#### عرض المستخدمين:
- قائمة شاملة لجميع المستخدمين المسجلين
- معلومات المستخدم: الاسم، البريد الإلكتروني، تاريخ التسجيل
- حالة المستخدم: نشط، محظور، معطل

#### حظر/إلغاء حظر مستخدم:
```
1. اختر المستخدم
2. انقر على "Block" أو "Unblock"
3. أدخل السبب (اختياري)
4. أكد الإجراء
```

#### عرض نشاطات المستخدم:
```
1. انقر على اسم المستخدم
2. سيظهر سجل النشاطات الخاص به
3. يمكنك تصفية حسب التاريخ أو نوع النشاط
```

### 4. إدارة حسابات Telegram (Accounts)

#### عرض الحسابات:
- قائمة جميع حسابات Telegram المتصلة
- معلومات الحساب: رقم الهاتف، اسم المستخدم، الحالة
- آخر استخدام

#### حالات الحساب:
- **Active**: الحساب نشط وجاهز للاستخدام
- **Limited**: الحساب محدود (قد يكون بسبب حظر من Telegram)
- **Banned**: الحساب محظور من Telegram
- **Inactive**: الحساب غير نشط

#### إجراءات على الحسابات:
```
1. فصل الحساب: "Disconnect"
2. إعادة تفعيل الحساب: "Reactivate"
3. تحديث الحالة: "Refresh Status"
```

### 5. سجل النشاطات (Activity Logs)

#### عرض السجلات:
- قائمة شاملة لجميع العمليات والأحداث
- معلومات: المستخدم، نوع العملية، الحالة، الوقت

#### الفلترة:
```
- حسب المستخدم
- حسب نوع العملية
- حسب الحالة (نجح، فشل)
- حسب التاريخ
```

#### تصدير السجلات:
```
1. اختر النطاق الزمني
2. انقر على "Export"
3. اختر الصيغة (CSV, PDF)
4. سيتم تحميل الملف
```

### 6. إدارة البروكسيات (Proxies)

#### إضافة بروكسي جديد:
```
1. انقر على "Add Proxy"
2. أدخل المعلومات:
   - الاسم
   - النوع (HTTP, HTTPS, SOCKS5)
   - المضيف
   - المنفذ
   - اسم المستخدم (اختياري)
   - كلمة المرور (اختياري)
3. انقر على "Add"
```

#### اختبار البروكسي:
```
1. اختر البروكسي
2. انقر على "Test"
3. سيظهر وقت الاستجابة والحالة
```

#### حذف البروكسي:
```
1. اختر البروكسي
2. انقر على "Delete"
3. أكد الحذف
```

### 7. الإعدادات (Settings)

#### الإعدادات العامة:
- **Max Accounts Per User**: الحد الأقصى لحسابات Telegram لكل مستخدم
- **Max Operations Per Day**: الحد الأقصى للعمليات يومياً
- **Session Timeout**: مدة انتهاء الجلسة (بالساعات)

#### قواعد Anti-Ban:
- **Min Delay**: الحد الأدنى للتأخير بين العمليات (ملي ثانية)
- **Max Delay**: الحد الأقصى للتأخير (ملي ثانية)
- **Max Per Hour**: الحد الأقصى للعمليات بالساعة
- **Max Per Day**: الحد الأقصى للعمليات يومياً

#### حفظ الإعدادات:
```
1. عدّل الإعدادات المطلوبة
2. انقر على "Save Settings"
3. سيظهر تأكيد بنجاح الحفظ
```

## الإشعارات والتنبيهات

### أنواع التنبيهات:
- **تحذير**: التراخيص التي تنتهي قريباً
- **خطأ**: حسابات محظورة أو معطلة
- **نجاح**: عمليات مكتملة بنجاح

### إدارة التنبيهات:
```
1. انقر على جرس الإشعارات
2. اعرض التنبيهات الحديثة
3. اضغط على التنبيه لعرض التفاصيل
```

## التقارير والتحليلات

### أنواع التقارير:
1. **تقرير المستخدمين**: نشاط وإحصائيات المستخدمين
2. **تقرير التراخيص**: حالة وأداء التراخيص
3. **تقرير العمليات**: تفاصيل العمليات والأخطاء
4. **تقرير الأداء**: أداء النظام والسرعة

### تنزيل التقارير:
```
1. اختر نوع التقرير
2. اختر النطاق الزمني
3. انقر على "Generate Report"
4. اختر الصيغة (PDF, Excel)
5. سيتم التحميل تلقائياً
```

## الأمان والصلاحيات

### مستويات الوصول:
- **Admin**: وصول كامل لجميع الميزات
- **Manager**: إدارة المستخدمين والتراخيص
- **Viewer**: عرض فقط بدون تعديل

### تغيير كلمة المرور:
```
1. انقر على صورة الملف الشخصي
2. اختر "Change Password"
3. أدخل كلمة المرور الحالية
4. أدخل كلمة المرور الجديدة
5. أكد كلمة المرور الجديدة
6. انقر على "Update"
```

### تفعيل المصادقة الثنائية:
```
1. انقر على "Security Settings"
2. اختر "Enable 2FA"
3. امسح رمز QR بتطبيق المصادقة
4. أدخل الرمز المولد
5. احفظ رموز الاسترجاع
```

## استكشاف الأخطاء

### مشكلة: لا يمكن تسجيل الدخول
```
1. تحقق من بيانات الاعتماد
2. أعد تعيين كلمة المرور
3. تحقق من اتصالك بالإنترنت
4. امسح ذاكرة التخزين المؤقت للمتصفح
```

### مشكلة: الصفحة بطيئة
```
1. أعد تحميل الصفحة
2. امسح ذاكرة التخزين المؤقت
3. جرب متصفح مختلف
4. تحقق من سرعة الإنترنت
```

### مشكلة: لا تظهر البيانات
```
1. تحقق من اتصالك بالإنترنت
2. أعد تحميل الصفحة
3. تحقق من صلاحياتك
4. اتصل بالدعم الفني
```
